# Carify

Carify est une application web basée sur Laravel pour gérer les voitures et les trajets.

## Installation

```bash
composer install
npm install
```

## Configuration

Copier le fichier `.env.example` en `.env` et générer la clé de l'application :

```bash
php artisan key:generate
```

## Base de données

Exécuter les migrations :

```bash
php artisan migrate
```

## Lancement de l'application

```bash
php artisan serve
```

## Technologies utilisées

- Laravel 11
- PHP 8.2+
- Tailwind CSS
- Vite
